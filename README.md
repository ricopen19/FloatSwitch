# FloatSwitch

macOS 用のフローティングタブバー型ウィンドウスイッチャー。

画面端に常駐するミニマルなバーから、起動中のアプリや Finder フォルダにワンクリックで切り替えられます。`Command + Tab` の視覚的な代替として、マウス移動を最小限に抑えた作業フローを実現します。

## 特徴

- **常に最前面に表示** — 他のウィンドウに隠れないフローティングバー
- **Dock 風拡大エフェクト** — ホバー時にアイコンが滑らかに拡大
- **アプリ + フォルダの 2 段表示** — 上段に起動中アプリ、下段に Finder で開いているフォルダ
- **ドラッグ移動** — 画面内の好きな位置に配置可能
- **横置き / 縦置き切り替え** — 右クリックメニューから選択
- **グローバルホットキー** — 修飾キー + 数字（1〜9）でバー左から順に切り替え
- **外観カスタマイズ** — グラデーション背景、透明度、バーサイズの調整
- **アプリの表示管理** — 不要なアプリを非表示にしたり、表示順序をドラッグで並び替え

## 必要環境

- macOS 14.0 以降
- アクセシビリティ権限（ウィンドウ操作に必要）

## インストール

### DMG からインストール

1. [Releases](../../releases) から最新の `.dmg` をダウンロード
2. DMG を開き、`FloatSwitch.app` を `Applications` フォルダにドラッグ
3. 初回起動時にアクセシビリティ権限を許可

### ソースからビルド

```bash
git clone https://github.com/ricopen/FloatSwitch.git
cd FloatSwitch
./scripts/build_dmg.sh
```

> ビルドには Xcode と `create-dmg`（`brew install create-dmg`）が必要です。

## 使い方

### 基本操作

| 操作 | 動作 |
|------|------|
| アイコンをクリック | そのアプリ / フォルダにフォーカス |
| アイコンを右クリック | ウィンドウ選択メニュー / バーから隠す |
| バーをドラッグ | 位置を移動 |
| バーを右クリック | 設定メニュー（向き・サイズ・表示管理など） |

### 設定

右クリックメニューの「設定...」から以下を変更できます：

- **外観** — ホバー時グラデーション（プリセット色 + 強度）、背景透明度
- **ホットキー** — アプリ / フォルダ切り替えの修飾キー

## 技術スタック

| 項目 | 技術 |
|------|------|
| 言語 | Swift |
| UI | SwiftUI + AppKit |
| ウィンドウ管理 | NSPanel（floating level） |
| 状態管理 | @Observable |
| アプリ検出 | NSWorkspace + Accessibility API |
| フォルダ検出 | ScriptingBridge（Finder） |
