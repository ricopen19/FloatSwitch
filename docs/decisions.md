# FloatSwitch 設計判断記録（軽量 ADR）

---

## ADR-001: 3次元回転・変形の見送り

- **日付**: 2026-03-06
- **決定**: 3次元回転（X / Y / Z 軸）・任意角度への傾斜は実装しない
- **理由**: 実装の複雑さに対してユーザー体験への効果が薄い。横置き・縦置きの切り替えで十分
- **代替**: 右クリックメニューから横/縦の向きを切り替える方式を採用済み

---

## ADR-002: グラデーション背景の実装方式

- **日付**: 2026-03-06
- **決定**: プリセット色 + 強度スライダー方式を採用
- **検討した選択肢**:
  - カラーピッカー（自由度が高すぎて迷う）
  - プリセットのみ（濃さの調整ができない）
  - プリセット + スライダー（採用: シンプルかつ柔軟）
- **表示タイミング**: ホバー時のみ（常時表示は視覚的にうるさい）
- **設定 UI**: SettingsView の「外観」セクション

---

## ADR-003: 外観設定の永続化方式

- **日付**: 2026-03-06
- **決定**: UserDefaults に JSON で保存（`appAppearance_v1`）
- **対象**: barSize, orientation, position, gradientPreset, gradientIntensity, showResidentApps
- **理由**: HotkeySettings と同じパターンで統一。各プロパティの `didSet` で自動保存、`init` で読み込み
