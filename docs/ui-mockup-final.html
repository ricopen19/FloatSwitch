縦表示のポイント:
・ガラスの光沢は左辺に（縦バーなので自然な光の向き）
・ドットはアイコン右側に配置（macOS Dock と同じ位置感覚）
・ラベルはホバー時に左にポップアップ（tooltip 風）
・ホバーで左方向に拡大（画面端に寄せるため）
・pill 形状とスプリング膨張は横と共通

実装で変わるポイント

FloatingBarShape ∩カーブ廃止 → RoundedRectangle(cornerRadius: 24) に統一
背景レイヤー インナーシャドウ + 上辺ハイライトストローク追加
レイアウト VStack(上段/下段) → HStack(apps | divider | folders) に変更
ホバーアニメ padding を Spring で増減（膨張エフェクト）
ラベル表示 常時表示 → 個別ホバー時のみ表示に変更
新規: ドット AppItem にアクティブ状態を持たせ、ドット描画を追加
