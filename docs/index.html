主な機能

シンプルだけど、ちゃんと使える。

📌

常に最前面に表示

他のウィンドウに隠れないフローティングバー。いつでもすぐにアクセスできます。

🔍

Dock 風の拡大エフェクト

ホバーでアイコンがスプリングアニメーションで滑らかに拡大。直感的に操作できます。

📂

アプリ + フォルダの 2 段表示

起動中アプリと Finder で開いているフォルダを一覧表示。すべてワンクリックで切り替え。

⌨️

グローバルホットキー

修飾キー + 数字（1〜9）で左から順にアプリを切り替え。マウスを動かす必要なし。

🎨

外観カスタマイズ

5 色のグラデーションプリセット、透明度調整、バーサイズの変更。自分好みに設定可能。

↔️

縦横レイアウト対応

横置き・縦置きを右クリックで切り替え。ドラッグで好きな位置に配置できます。

Core Feature

ワンクリックでウィンドウ切り替え

バー上のアイコンをクリックするだけで、対象アプリのウィンドウが最前面に。最小化されたウィンドウも自動で復元します。複数ウィンドウがある場合はサブメニューから選択できます。

Customization

ドラッグで自由に並び替え

よく使うアプリを左に、たまに使うアプリを右に。アイコンをドラッグするだけで表示順序を変更でき、並び順は自動保存されます。

使い方

1

インストール

DMG をダウンロードして、アプリケーションフォルダにドラッグ。初回起動時にアクセシビリティ権限を許可します。

2

バーが表示される

画面端にフローティングバーが常駐。起動中のアプリと開いているフォルダが自動的に並びます。

3

クリックで切り替え

アイコンをクリック、またはホットキーで瞬時に切り替え。もう Cmd+Tab を連打する必要はありません。

5 つのカラープリセット

気分やデスクトップに合わせて、バーの雰囲気を変えられます。
