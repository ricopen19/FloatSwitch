## フェーズ 6: 表示アプリのカスタマイズ

> 背景: ウィンドウを持たない常駐アプリ（VLC・Nani・Voiceink 等）の自動排除を試みたが
> 動作が不安定化したため、ユーザー指定方式に切り替える。

- [ ] 非表示アプリリストを JSON ファイルで管理する（`~/.config/floatswitch/hidden_apps.json`）
- [ ] アイコンを右クリック → 「バーから隠す」でリストに追加・バーから即時除外する
- [ ] 右クリックメニュー（バー全体）に「隠したアプリを管理...」項目を追加し、再表示できるようにする
- [ ] アプリの表示順序をユーザーが指定できるようにする（JSON で bundleID 順序を定義、未指定は起動順）
